Формат файла res.xlsx:
=========================

Файл должен содержать 3 колонки (в любом порядке):

1. ФИО - полное имя (обязательно)
2. Факультет - название факультета (обязательно)
3. telegram_username - юзернейм в Telegram (обязательно для рассылки)

Пример содержимого:

| ФИО                      | Факультет       | telegram_username  |
|--------------------------|-----------------|-------------------|
| Иванов Иван Иванович     | Информатики     | @ivanov_ivan      |
| Петрова Анна Сергеевна   | Экономики       | petrov_anna       |
| Сидоров Петр Петрович    | Медицины        | @sidorov_petr     |
| Козлова Мария Ивановна   | Физики          | kozlova_maria     |

Важные замечания:
-----------------
✓ Символ @ в telegram_username может быть, а может не быть (оба варианта работают)
✓ Все telegram_username автоматически приводятся к нижнему регистру
✓ Если в строке нет ФИО - она будет пропущена
✓ Дубликаты по telegram_username будут пропущены
✓ Пустые telegram_username допускаются, но для рассылки нужен username

Пример в Excel:
---------------
A1: ФИО
B1: Факультет  
C1: telegram_username

A2: Иванов Иван Иванович
B2: Информатики
C2: @ivanov_ivan

A3: Петрова Анна Сергеевна
B3: Экономики
C3: petrov_anna

И так далее...

После создания файла:
---------------------
1. Сохраните файл как res.xlsx
2. Положите его в корень проекта ShaBot
3. Запустите: python scripts/load_reserv.py
4. Проверьте результат командой /poter в боте
